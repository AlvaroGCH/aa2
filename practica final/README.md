# Asistente RAG sobre TFG con n8n + MCP

Proyecto final de la asignatura **Aprendizaje Automático 2**.

Sistema de asistente inteligente basado en RAG (Retrieval-Augmented Generation)
que responde preguntas sobre un Trabajo Fin de Grado. El conocimiento se
expone a través de dos interfaces:

- **Bot de Telegram** (agente propio en n8n con GPT-4o-mini)
- **Claude Desktop** (vía servidor MCP implementado en n8n)
